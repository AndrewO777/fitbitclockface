# fitbitclockface
This is a fitbit clockface made for school

Make sure before you make any changes that you do a pull to avoid merge conflicts.

To run the clockface on your device or the simulator do this
1. Clone the repo, go into the fitbitclockface directory and run "npm install"
2. Start the simulator or connect your device to the machine.
3. run "npx fitbit-build"
4. run "npx fitbit"
5. run "install"
